# Volcano Watch

Student Assignment 2 web app for the Web App Development 1 module.

## Tech

- Node.js
- Express
- Handlebars
- JSON data store
- Express sessions
- Multer image uploads

## Pages

- / (Welcome)
- /signup
- /login
- /profile
- /dashboard
- /dashboard/:id
- /about
- /safety

## Assignment 2 Work Added

- Signup, login, and logout pages
- Basic password rules
- Dashboard only shows the logged-in user's collections
- Add and delete volcano collections
- Add, edit, and delete volcano items
- Search and sort on the dashboard and details pages
- App statistics and current user statistics
- Image upload for collections, items, and profile pictures
- Small `formatDate` helper for displaying dates
