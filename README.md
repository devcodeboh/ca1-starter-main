# Volcano Watch

Simple CA1 web app for the Web App Development module.

## Tech

- Node.js
- Express
- Handlebars
- JSON data store

## Pages

- / (Welcome)
- /dashboard
- /dashboard/:id
- /about
- /safety

## Run

npm install
npm start

Open http://localhost:3000

- models/activity-store.json
- models/app-store.json
